Cities Page

Welcome to Cities Page, RESTful API for :
  1. HTTP GET request method : get_all_cities() for displaying all the cities
  2. HTTP GET request method : get_city(name) for displaying a specific city
  3. HTTP POST request method : add_city() for adding a new city
  4. HTTP PUT request method : update_city(name) for updating an existing city
  5. HTTP DELETE request method : delete_city(name) for deleting/removing an existing city
